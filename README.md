# Expense Tracker 

live Demo-https://expense-tracker-app-six-indol.vercel.app/

A simple Expense Tracker built with HTML, CSS, and JavaScript to practice DOM manipulation, local storage, event handling, and data visualization.

## Features

* Add new expenses
* Edit existing expenses
* Delete expenses
* Categorize expenses
* Calculate total spending
* Store data using Local Storage
* Visualize spending by category with charts
* Responsive and beginner-friendly UI

## Tech Stack

* HTML
* CSS
* JavaScript (ES6)
* Local Storage API
* Chart.js

## What I Learned

While building this project, I practiced:

* DOM manipulation
* Event handling and event delegation
* Working with arrays and objects
* Array methods (`forEach`, `filter`, `reduce`)
* Dynamic UI rendering
* Local Storage for data persistence
* CRUD operations (Create, Read, Update, Delete)
* Basic data analytics and chart visualization
* Organizing JavaScript code into reusable functions

## Future Improvements
